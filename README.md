🛍️ Online Clothing Store (React + Node API)

A full-stack e-commerce web application built with React (Vite) frontend and Node.js + Express API (deployed on Render).
The project simulates a real online clothing shop with product browsing, filtering, cart system and checkout flow.


🚀 Live Demo

https://xdqwe-c7cv.vercel.app/

⚙️ Tech Stack

Frontend:
React 19 (Vite)
React Router DOM
React Hooks (useState, useEffect, useContext)
React Icons
CSS (custom styling)
LocalStorage (cart persistence)

Backend:
Node.js
Express.js
CORS middleware
JSON-based data (db.json)

Deployment:
Vercel (frontend)
Render (backend API)

📦 Features

🛒 Shopping system

Add products to cart
Remove products from cart
Cart total price calculation
Persistent cart (localStorage)

👕 Product system

Fetch products from API
Product details page
Image gallery slider
Product categories (shoes, hoodies, jeans, etc.)

🔍 Filtering & sorting

Filter by gender (Man / Woman)
Filter by category
Sort by:
price (low → high / high → low)
alphabetically

💳 Checkout

Address input form
Order confirmation message
Cart reset after purchase

🌐 UI / UX

Responsive layout
Dynamic navbar cart counter
Conditional rendering (empty cart state, success message)
Icons from React Icons

🧠 State Management

React Context API (Gender selection)
Local component state (useState)
Side effects (useEffect for API + localStorage sync)

📌 Future improvements

Authentication (login/register)
Payment integration (Stripe)
Product database (MongoDB instead of JSON)
Admin panel (add/edit products)
Better UI animations
